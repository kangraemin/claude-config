# Library Lint

`~/.claude/.claude-library/library/` 의 건강 상태를 점검하고 문제를 수정한다.

## 체크 항목

### 1. 관련 태그 양방향 검증
- 모든 `index.md`에서 `관련:` 태그를 파싱
- A→B가 있으면 B→A도 있어야 함
- **깨진 것은 바로 수정한다**

### 2. index.md 파일 목록 검증
- 주제 폴더 내 `.md` 파일 목록과 `index.md`의 `지식 목록` 비교
- 등록 안 된 파일이 있으면 index.md에 추가

### 3. LIBRARY.md 카운트 검증
- 각 주제의 괄호 안 숫자 `(N)`이 실제 지식 파일 수와 일치하는지
- 불일치하면 수정

### 4. 고아 파일
- 어떤 `index.md`에도 안 나오는 `.md` 파일
- 보고 후 해당 index.md에 추가

### 5. 오래된 항목 체크
- 날짜가 3개월+ 된 항목 목록 출력
- "아직 유효한가?" 판단은 사용자에게 위임 (자동 삭제 안 함)

### 6. 크로스레퍼런스 제안
- `관련:` 태그가 없는 주제 목록 출력
- 내용 기반으로 연결할 수 있는 주제 쌍 제안
- 제안만 하고 사용자 확인 후 적용

## 플로우

1. **스캔**: 위 6개 항목 전부 체크
2. **자동 수정**: 1~4번은 확인 없이 바로 수정
3. **보고**: 수정한 것 + 수동 확인 필요한 것(5~6번) 출력
4. **CHANGELOG**: 수정 사항 `CHANGELOG.md`에 append
5. **커밋**: git commit & push

## 출력 형식

```
🔧 자동 수정:
- [항목] 설명

⚠️ 수동 확인 필요:
- [항목] 설명

📊 통계:
- 전체 주제: N개
- 전체 지식 파일: N개
- 관련 태그 연결: N개
- 미연결 주제: N개
```
