# Jazz Weather Recommender

현재 서울 날씨를 조회한 뒤, 그 날씨의 정서와 어울리는 재즈곡 3~5곡을 YouTube 링크와 함께 추천한다.

## 작업 흐름

### 1단계: 서울 현재 날씨 조회

웹 검색으로 서울 현재 날씨를 가져온다. 다음 정보를 파악한다:

- **기온** (°C)
- **날씨 상태**: 맑음 / 흐림 / 비 / 눈 / 안개 / 황사 등
- **체감 온도** (있으면)
- **시간대**: 낮/밤 (현재 시간 기준)

검색어 예시: `서울 현재 날씨` 또는 `Seoul weather now`

### 2단계: 날씨 → 재즈 분위기 매핑

날씨 상태와 온도/시간대를 종합해 음악적 정서를 결정한다.

| 날씨 / 조건 | 음악적 정서 | 재즈 방향 |
|---|---|---|
| 맑고 따뜻함 (15°C↑) | 밝고 경쾌 | 스윙, 보사노바, 쿨재즈 |
| 맑고 쌀쌀함 (10°C↓) | 청명하고 고요 | 피아노 트리오, 챔버재즈 |
| 흐림 / 구름 많음 | 차분하고 사색적 | 모달재즈, 포스트밥 |
| 비 (약한 비) | 감성적, 내향적 | 발라드, 스탠더드 |
| 비 (강한 비/폭우) | 강렬하고 역동적 | 하드밥, 프리재즈 |
| 눈 | 고즈넉하고 서정적 | 피아노 솔로, 윈터재즈 |
| 안개 / 흐린 밤 | 신비롭고 몽환적 | 앰비언트재즈, 퓨전 |
| 무더위 (30°C↑) | 에너지 넘침 | 라틴재즈, 펑키재즈 |
| 봄 / 선선한 오후 | 여유롭고 낭만적 | 보사노바, 쿨재즈 |

단순히 표를 기계적으로 적용하지 말고, 날씨의 복합적인 뉘앙스를 반영한다.  
예: "흐리고 5°C인 저녁"은 차갑고 쓸쓸한 정서 → 느린 발라드나 마일스 데이비스 스타일의 쿨재즈가 어울림.

### 3단계: 재즈곡 선정

3~5곡을 선정한다. 선정 기준:

- 날씨의 **정서/분위기**와 실제로 어울리는 곡이어야 한다 (장르 레이블만 맞추는 것 금지)
- 유명한 스탠더드부터 덜 알려진 명곡까지 다양하게 섞는다
- 같은 아티스트가 2곡 이상 겹치지 않도록 한다
- 각 곡이 **왜** 지금 날씨와 어울리는지 구체적으로 설명할 수 있어야 한다

### 4단계: YouTube 링크 검색

각 곡에 대해 웹 검색으로 YouTube 공식 영상 또는 고음질 업로드 링크를 찾는다.

검색어 패턴: `[곡명] [아티스트명] youtube official` 또는 `[곡명] [아티스트명] site:youtube.com`

- 공식 채널 영상을 우선한다
- 못 찾으면 `https://www.youtube.com/results?search_query=...` 형태의 검색 링크로 대체한다

### 5단계: 결과 출력

아래 형식으로 출력한다.

---

## 출력 형식

```
## 🎷 지금 서울 날씨에 어울리는 재즈

**현재 날씨**: [날씨 상태], [기온]°C ([시간대])
**오늘의 정서**: [한 문장으로 분위기 묘사]

---

### 1. [곡명] — [아티스트]
> [앨범명, 연도] · [서브장르]

[이 날씨와 어울리는 이유를 2~3문장으로. 곡의 질감/분위기/특징을 날씨와 연결해서 설명한다.]

▶ [YouTube 링크]

---

### 2. ...

(3~5곡 반복)
```

**주의사항:**
- 이모지는 헤더 한 곳에만 쓴다
- 설명은 "이 곡은 ~합니다" 같은 백과사전 투가 아니라, 실제로 날씨를 경험하는 사람에게 말하듯 쓴다
- YouTube 링크가 검색으로 확인된 것이면 바로 제시, 불확실하면 검색 링크를 안내한다
